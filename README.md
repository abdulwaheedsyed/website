## Quick Start

1. Clone the repository
2. Open `index.html` in your web browser
3. Or use a local server:
   ```bash
   python3 -m http.server 8080
   ```
4. Visit `http://localhost:8080` in your browser

## Customization

### Profile Information

Edit `index.html` to customize:

- **Name and Title**: Update the `<h1>` and tagline in the hero section
- **Profile Photo**: Replace the placeholder image URL in the hero section
- **About Me**: Update the paragraphs in the About section
- **Skills**: Modify the skill categories and items in the Skills section
- **Experience**: Update job titles, companies, dates, and responsibilities
- **Education**: Update degree, university, and certifications
- **Contact**: Update email address and location

### Social Media Links

Update the social media URLs in the hero and contact sections:
- GitHub: `https://github.com/abdulwaheedsyed`
- LinkedIn: `https://linkedin.com/in/abdulwaheedsyed`
- Twitter: `https://twitter.com/abdulwaheedsyed`
- Email: `contact@sawaheed.com`

### Colors and Styling

Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --text-color: #333333;
    --text-light: #666666;
    --bg-light: #f8fafc;
    --bg-white: #ffffff;
}
```
